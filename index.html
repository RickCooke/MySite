Experience

  • The Boeing Company

    Mobile Software Developer

    This past summer I was working as a software developer for Boeing's Mobile Solutions group in Seattle. During my time here I was given challenge of developing two entirely different apps, both of which required my undivided attention. The challenge was matched with a strong desire to learn and code as much as I could. One of the projects was a mapping application that would allow users to find any company building and use google maps' API to provide turn-by-turn directions to it. Once at the building a user could then look at the conference rooms available in the building based on availability and features. A handy feature in an overcrowded office space. My other project is mostly proprietary but involved the dynamic drawing of custom UI components as well as the design of the supporting API. Both of these applications were developed for the Android platform using native Java and XML.

    Program Manager
